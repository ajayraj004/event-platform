# Event Scraping & Management Platform

A full-stack web application that demonstrates event ingestion, lifecycle tracking, and administrative review.

## Features
- Public event listing with email capture
- Admin dashboard to review and import events
- Event lifecycle: new, updated, inactive, imported
- Simulated scraping pipeline

## Tech Stack
- Frontend: React + Vite
- Backend: Node.js + Express
- Database: MongoDB Atlas
- Deployment: AWS EC2 (backend), AWS S3 (frontend)
